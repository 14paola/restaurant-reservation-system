# Restaurant Reservation System - Frontend

Frontend desarrollado con Vue 3 y Vite para consumir la API de reservas del restaurante.

## Tecnologías usadas

- Vue 3
- Vite
- Axios
- Vue Router

## Funcionalidades implementadas

- Formulario para crear reserva
- Consulta de disponibilidad
- Vista de confirmación de reserva
- Cancelación de reserva por código

## Instalación

```bash
npm install
npm run dev
